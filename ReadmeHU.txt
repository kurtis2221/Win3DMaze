|Windows 3D Maze Game - Kurtis által (2021)|
\==========================================/

=======================
Minimális rendszerigény
=======================
- Windows 7 vagy újabb
- .Net Framework 4.5
- 1024 MB RAM
- 2.0 Ghz CPU
- Nvidia 9500 GT
- 60 MB szabad terület
- 30 MB telepítés után

=============
Hibaelhárítás
=============
Ha a játék nem menti a beállításaidat, ellenőrizd a játék mappa jogosultságait
vagy indítsd a játékot rendszergazdaként.

====
Info
====
Ugyanaz a játék mint az eredeti képernyőkímélő.
A cél elérni a kijáratot jelölő elemet.
Néhány tárgy felfordít.

===========
Beállítások
===========
Setup - Labirintus kinézet
Control - Irányítás típusa
FOV - Field of View (látószög mérete)
Mouse sens. - Egérérzékenység

=========
Irányítás
=========

Modern
======
Nyílbillentyűk / W,A,S,D - Mozgás
Egér - Nézelődés

Classic
=======
Nyílbillentyűk / W,A,S,D - Mozgás
Alt - Oldalazás

Original
========
Nyílbillentyűk / W,A,S,D - Mozgás

Egyéb
=====
Shift - Futás
TAB - Térkép be/ki
Num + / - - Térkép nagyítás
N - Új pálya
ESC - Vissza a menübe

========
Csalások
========
Backspace + I - Megfordítás
Backspace + O - Falon átjárás be
Backspace + P - Falon átjárás ki
